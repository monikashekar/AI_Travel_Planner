Travel Planner – AI-Powered Trip Planning Platform

This project is a scalable travel planning platform designed using a microservice-oriented backend architecture. The system integrates AI-powered itinerary generation, real-time location search, and performance optimization techniques to deliver a responsive travel planning experience.

Key Features

Built a scalable backend using NestJS, PostgreSQL, and Redis

Implemented AI-generated travel itineraries using OpenAI APIs

Developed a search and recommendation engine integrating Google Places

Designed an event-driven architecture using BullMQ job queues

Improved API performance using Redis caching

Created a React dashboard for trip visualization, itinerary management, and budgeting
